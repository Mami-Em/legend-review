About

Overview

This project is an anime review website inspired by Harvard CS50 project. Users can register and login with their Google account, search for any anime, see reviews and ratings made by others, and can leave one review per anime. See below the technologies used:

PostgreSQL

A PostgreSQL database is hosted on Railway, an online web and application hosting service, to store all the data. There are only two tables since this project will be using free tier, one to keep track of the reviews and ratings and one to keep track of the users.

Python and Flask

Flask is a microframework for building web applications with Python. Flask and Python will manage all the server side of this web application.

Kitsu API

Kitsu is a modern anime discovery, its API is used to track all anime seen here. The ratings and reviews displayed to user will be a combination of the ones from the Railway database and the ones accessed from this API.

Auth0

This platform will authenticate and authorize user to login or register with their Google account for this application.

Tailwind UI and Figma

The logo was built from scratch using Figma and thanks to Tailwind UI, components found there are easy to customize and fasten the frontend work.

Render

The code from github is auto deployed to render, which is a simple and fast way to host web applications and many other services.
